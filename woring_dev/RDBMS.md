# RDBMS study

```note
## RDBMS란?

  Relational Data Base Manage System
  관계형 데이터 베이스 관리 시스템
  단순히 관련성이 있는 것끼리 묶는것을 넘어서 다른 데이터와의 연결성을 만들 수도 있다. 

  DBMS란?
  DataBase Manage Service 
  데이터를 관리 하는 서비스, 즉 소프트웨어를 말한다. 

```
